Slide Sync

Slide Sync appears to be a cutting-edge project that leverages gesture control for slide presentations, enhancing the ease of use during live presentations. Utilizing machine learning and computer vision, it allows presenters to seamlessly navigate through slides with simple hand gestures. The project's front end is crafted with HTML, CSS, and JavaScript, ensuring a responsive and user-friendly interface, while the back end is powered by Django, a robust and scalable Python web framework. This combination of technologies suggests a highly interactive and efficient tool for modern presentations.
